# chanyeong-server

Back-End Server for the chanyeong webpage on chanyeong. GraphQL, Typescript, NodeJS

## Resolver

### Lock Resolvers:

- [ ]  admin Sign Up
- [ ]  Edit User Info

### Authentication:

- [ ]  Generate JWT ( Refresh and Access )
- [ ]  Generate JWT ( only Access )
- [ ]  Verify Access JWT

### Public Resolvers:

- [ ]  Sign In
- [ ]  Get FrontEnd Skills
- [ ]  Get BackEnd Skills
- [ ]  Get DebOps Skills
- [ ]  Get Group Projects
- [ ]  Get Personal Projects
- [ ]  Get All Posts
- [ ]  Get Category Posts

### Private Resolvers:

- [ ]  Add Skill
- [ ]  Eidt Skill
- [ ]  Add Project
- [ ]  Eidt Project
- [ ]  Write Post
- [ ]  Edit Post
