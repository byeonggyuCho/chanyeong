# chanyeong-server

Back-End Server for the chanyeong webpage on chanyeong. GraphQL, Typescript, NodeJS

## Resolver

### Lock Resolvers:

- [x]  admin Sign Up
- [x]  Edit User Info

### Authentication:

- [x]  Generate JWT ( Refresh and Access )
- [x]  Generate JWT ( only Access )
- [x]  Verify Access JWT

### Public Resolvers:

- [x]  Sign In
- [x]  Get Skills
- [x]  Get Group or Personal Projects
- [x]  Get Picked Projects
- [x]  Get Posts
- [x]  Get Picked Posts
- [x]  Get Tag Posts

### Private Resolvers:

- [x]  LogOut
- [x]  Add Skill
- [x]  Eidt Skill
- [x]  Delete Skill
- [x]  Add Project
- [x]  Eidt Project
- [x]  Delete Project
- [x]  Add Tag
- [x]  Write Post
- [x]  Edit Post
- [x]  Delete Post
