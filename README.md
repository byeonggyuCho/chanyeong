# Chanyeong

개발자 조찬영의 포트폴리오 및 블로그 기능이 포함된 웹 애플리케이션이다.

Toast서비스에 배포하였으며 아직 개발중인 상태이다.

기능 구현은 각 폴더(back, front)의 README파일에 구현된 기능이 To-do 양식으로 나타내진다.
